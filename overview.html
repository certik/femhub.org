Our mission: To create an open-source distribution featuring many finite element codes, along with a web notebook and a unified Python interface. We want FEMhub to become an alternative to commercial FEM codes.

Mailing list: Please use this mailing list for all communications related to FEMhub.

How to join: See a list of nice project ideas and a list of simple-to-fix issues related to FEMhub.

The FEMhub project

FEMhub is an open source distribution of finite element software. It aims to provide a unified (Python) interface to all of them and also to allow to use them over the web notebook. Currently it contains the project below.
