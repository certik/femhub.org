Our mission: To create an open-source distribution featuring many finite element codes, along with a web notebook and a unified Python interface. We want FEMhub to become an alternative to commercial FEM codes.

Mailing list: Please use this mailing list for all communications related to FEMhub.

How to join: See a list of nice project ideas and a list of simple-to-fix issues related to FEMhub.

FEM codes and resources

hp-FEM codes: Besides Hermes, there are very few hp-FEM codes we are aware of:

  • Phaml (developed by William Mitchell at NIST)
  • Concepts (developed by Ph.D. students at SAM, ETH Zurich)
  • 1Dhp90, 2Dhp90, 3Dhp90 (developed by L. Demkowicz at UT Austin)
If you know of any other opensource adaptive hp-FEM code that should be here, please let us know!

Standard FEM codes: There are numerous (standard low-order) FEM codes available on the web. We only list some of them:

  • SfePy
  • libMesh
  • deal.II
  • FEniCS Project
  • OOFEM
  • NGSolve
If you know of any other opensource FEM code that should be here, please let us know!

Internet FEM resources:

  • Internet Finite Element Resource (IFER) -- conferences and courses, newsgroups, homepages, tutorials, libraries, mesh generation and visualization tools, books, etc.
  • Wikipedia list of FEM packages
  • Finite Element People
If you know of any other quality sites worth including here, please let us know too!
