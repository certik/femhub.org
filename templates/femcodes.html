Other FEM codes

hp-FEM codes: Besides Hermes, there are very few hp-FEM codes we are aware of:

  • Concepts (developed by Ph.D. students at SAM, ETH Zurich)
  • 1Dhp90, 2Dhp90, 3Dhp90 (developed by L. Demkowicz at UT Aust in)
If you know of any other opensource adaptive hp-FEM code that should be here, please let us know!

Standard FEM codes: There are numerous (standard low-order) FEM codes available on the web. We only list some of them:

  • libMesh
  • SfePy
  • deal.II
  • FEniCS Project
  • OOFEM
If you know of any other opensource FEM code that should be here, please let us know!

Miscellaneous internet FEM resources:

  • Internet Finite Element Resource (IFER) -- conferences and courses, newsgroups, homepages, tutorials, libraries, mesh generation and visualization tools, books, etc.
  • Wikipedia list of FEM packages
  • Finite Element People
If you know of any other quality sites worth including here, please let us know too! {% endblock %}
