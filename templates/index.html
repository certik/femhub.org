The FEMhub project

Our mission: To create an open-source distribution featuring many finite element codes, along with a web notebook and a unified Python interface. We want FEMhub to become an alternative to commercial FEM codes.

Installation

Download, unpack, build: 

$ wget http://sage.math.washington.edu/home/ondrej/scratch/femhub/femhub-0.9.5.tar
$ tar xf femhub-0.9.5.tar
$ cd femhub-0.9.5
$ make

Depending on the speed of your computer, wait between 37 minutes to 1.5 hour. That's it.

If you want, you can also download a binary from here, however, if it doesn't work for you, compile from source, that should always work (if not, please report a bug).

First Steps

$ ./spd
----------------------------------------------------------------------
| Femhub (FEM Distribution), Version 0.9.5, Release Date: 2009-07-14 |
| Type notebook() for the GUI.                                       |
----------------------------------------------------------------------
In [1]: notebook()
and a browser will start with the web notebook. Follow the instructions here to do your first calculation.
