Hermes project

Hermes is a free C++/Python library for rapid prototyping of adaptive FEM and hp-FEM solvers available under the GPL license (Version 2, 1991). The library has a clean design and modular structure. The following unique features make Hermes different from other FEM and hp-FEM codes:

  • PDE-independent adaptivity algorithms: Hermes does not use any analytic information about the underlying PDE to guide automatic adaptivity. This means that any PDE that can be solved using finite elements, can be solved in Hermes using automatic hp-adaptivity. Because of this, Hermes can solve adaptively arbitrary combinations of multiphysics PDE systems.
  • Arbitrary-level hanging nodes: This technique makes element refinements completely local. In other words, the refinement of an element in the mesh never causes its neighbors to be refined. This greatly simplifies adaptive algorithms.
  • Adaptive multimesh hp-FEM: This technique makes it possible to approximate various solution components (such as various physical fields or various eigenfunctions) on individual meshes equipped with mutually independent adaptivity mechanisms. Still, the discretization is monolithic and there is no projection error between meshes. This has a huge positive impact on the efficiency and accuracy of multiphysics, eigenvalue, and other computations involving multiple solution components.
  • Space-time adaptive hp-FEM on dynamical meshes: So far, Hermes is the only hp-FEM code capable of space-time hp-adaptivity for time-dependent problems. Our novel technique is based on a combination of the multimesh hp-FEM with the classical Rothe's method. See sample computational videos for illustration.
