Welcome to the FEMhub Project

There are many free finite element (FEM) codes on the web, whose comparison and interoperability are extremely difficult due to different installation requirements, different input and output formats, and capabilities. The goal of FEMhub is to bring transparency and establish basic standards in the development of open dourse FEM codes. All codes in FEMhub will be easy to compare via a unified Python interface and available to anyone for free either as a desktop application or via an interactive web notebook. The distribution is developed by an open source community around the hp-FEM group at the University of Nevada, Reno and available under the GPL license (Version 2, 1991).

Project Goals

  • Make it possible to download and install many open source FEM codes easily.
  • Provide access to modeling, mesh generation, and visualization tools.
  • Allow the user to solve a model by means of various codes easily.
  • Make all codes in FEMhub usable online via an interactive web notebook.
  • Make it possible to run realistic FEM computations from any classroom.
  • Create a quality assessment procedure for open source FEM codes.
FEMhub is a joint effort with NIST. {% endblock %}
